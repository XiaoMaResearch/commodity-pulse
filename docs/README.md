# App Store Web Docs

These files are intended to be published with GitHub Pages so they can serve as the `Privacy Policy URL` and `Support URL` in App Store Connect.

Recommended publishing path:

1. In GitHub, open the `commodity-pulse` repository settings.
2. Enable GitHub Pages.
3. Publish from the `main` branch and `/docs` folder.
4. Confirm these URLs load:
   - `https://xiaomaresearch.github.io/commodity-pulse/privacy-policy.html`
   - `https://xiaomaresearch.github.io/commodity-pulse/support.html`
